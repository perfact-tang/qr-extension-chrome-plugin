
# プライバシーポリシー

このプライバシーポリシーは、[QR Code Master]（以下「本拡張機能」といいます）が、ユーザーの情報をどのように取り扱うかについて定めるものです。

## 1. データ収集の原則

**本拡張機能は、ユーザーのいかなるデータも収集しません。**

ユーザーの個人情報、ウェブ閲覧履歴、操作ログ、デバイス情報など、個人を特定できる情報（PII）を含むいかなる種類のデータも、ローカル環境に保存したり、外部のサーバーやサードパーティに送信したりすることは一切ありません。

## 2. ローカルデータの取り扱い (※拡張機能内で設定などを保存する場合に記載)

本拡張機能がユーザーの利便性のためにローカルに保存するデータがある場合、そのデータは以下の通りに取り扱われます。

* **保存場所**: データは、ユーザーのローカル環境（例: Chromeの`chrome.storage.local`や`chrome.storage.sync`など）に保存されます。
* **外部送信**: ローカルに保存されたデータは、本拡張機能が意図的に外部へ送信することはありません。
* **消去**: ユーザーが本拡張機能を削除すると、これらのローカルデータはブラウザによって消去されます。

## 3. データの利用目的

本拡張機能はデータを収集しないため、利用目的も存在しません。

## 4. 情報の共有・開示

本拡張機能はデータを収集しないため、ユーザーの情報を第三者と共有したり、開示したりすることはありません。

## 5. 外部サービスとの連携

本拡張機能は、データ収集やトラッキングを目的とした外部の解析サービス（例: Google Analytics、Firebaseなど）や広告サービスとは**一切連携していません**。

本ポリシーは、法令の変更または本拡張機能の機能変更（例: 将来的にデータ収集が必要になった場合など）に伴い改定することがあります。本ポリシーに重要な変更がある場合は、ユーザーに適切に通知します。
